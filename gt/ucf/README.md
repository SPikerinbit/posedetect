# UCF Annotation Interface

This directory reserves the manual annotation interface for the UCF preview set.

Files:
- `syq.csv`
- `szh.csv`
- `lrx.csv`
- `mapping_video_to_annotator.csv`
- `assignment_summary.json`
- `人工标注说明.md`

All three CSVs use the same column schema as the Penn interface. The current rows are placeholders for the selected preview videos; annotators only need to fill the temporal fields and the action label.
